## 📦 项目位置
`C:\cursor\qwen-demo\md-editor`

## 🎯 技术栈总结

| 组件 | 选型 | 说明 |
|------|------|------|
| **框架** | Electron + React 19 + TypeScript | 跨平台、类型安全、现代化 |
| **编辑器** | Monaco Editor | VS Code 同款，性能最优 |
| **解析器** | markdown-it | 高性能 Markdown 解析 |
| **构建工具** | Vite | 极速开发体验 |

## ⚡ 性能优化

1. **防抖解析** (150ms) - 避免频繁重绘
2. **Monaco 虚拟滚动** - 支持大文件
3. **Web Worker** - 不阻塞主线程
4. **按需渲染** - 只更新必要 DOM

## ✨ 已实现功能

- ✅ 实时预览（编辑/双栏/预览模式）
- ✅ 文件操作（新建/打开/保存）
- ✅ 暗色/亮色主题切换
- ✅ 导出 HTML
- ✅ 快捷键支持 (Ctrl+S/O/N)
- ✅ Monaco 完整语法高亮

## 🚀 快速开始

```bash
# 进入项目
cd md-editor

# 启动开发（浏览器模式）
npm run dev:react

# 访问 http://localhost:5173
```

**注意**: Electron 依赖可能需要科学上网下载。如果遇到问题，可以先使用浏览器模式开发。
